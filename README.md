## Config / Instalação

Criando um ambiente virtual

```bash
  python -m venv venv
  source venv/bin/activate
```

Rodando o Script de Dependencias

```bash
  ./setup.sh
```
    
## Rodando localmente

```bash
  uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```
